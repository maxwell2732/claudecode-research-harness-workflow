# Feedback Skill

A skill for implementing in-app feedback forms.
First check the applicability conditions in `../ui-skills.md` and `../frontend-design.md`.

---

## Trigger Phrases

- "Add a feedback feature"
- "I want to collect user feedback"
- "Create a bug report form"
- "Add a contact form"

---

## Features

- Feedback forms
- Bug reporting
- Feature requests
- Satisfaction surveys

---

## Execution Flow

1. Check the applicability conditions for the constraint set
2. Check project configuration
3. Choose the type of feedback
4. Create the form UI
5. Create the API endpoint
6. Configure the data storage destination
