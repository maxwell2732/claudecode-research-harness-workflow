# Auth Skill

A skill for implementing authentication using Clerk or Supabase Auth.

---

## Trigger Phrases

- "Add login functionality"
- "Add authentication"
- "Implement authentication with Clerk"
- "Set up Supabase Auth"
- "Add Google login"

---

## Features

- Sign-up / Login
- Social login (Google, GitHub)
- Email verification
- Password reset
- User profile management

---

## Execution Flow

1. Check project configuration
2. Select Clerk or Supabase Auth
3. Install required packages
4. Generate authentication configuration files
5. Create login/sign-up UI
6. Configure middleware/protected routes
