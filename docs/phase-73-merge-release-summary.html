最終判断

merge判断

CodeRabbitを必須gateにしていないなら、オーナー判断でmerge可能。release-gradeに寄せるなら、CodeRabbit pending が消えてからmerge。 Actions側の失敗は残っていない。

release判断

いま即公開releaseはしない。merge後に main で clean preflight を再確認し、version/tagを切って、 tag-triggered release workflow が preflight込みで通ったら release GO。
